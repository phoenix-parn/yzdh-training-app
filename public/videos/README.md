# 视频资源说明

## 📁 目录用途
此目录用于存放本地视频文件。

## 📹 视频文件列表
- `video-01.mp4` - 主体结构与内部装修安装相结合
- `video-02.mp4` - 预制芯叠合柱
- `video-03.mp4` - 预制肋叠合墙
- `video-04.mp4` - 预制叠合顶板及中板

## 📝 注意事项
1. 视频文件较大(约225MB),已添加到`.gitignore`中,不会提交到Git仓库
2. 在GitHub Actions构建APK时,需要确保这些视频文件存在
3. 视频文件来源: `E:\Project\PRJ06-山东重点研发-济南交投\07-项目资料\素材\工法\video`

## 🔄 如何获取视频文件
如果你克隆了这个仓库但没有视频文件,请:
1. 从原始项目目录复制视频文件到此目录
2. 或者联系项目管理员获取视频文件

## 🚀 GitHub Actions构建
为了让GitHub Actions能够构建包含视频的APK,需要:
1. 将视频文件上传到GitHub Releases或其他存储服务
2. 在构建脚本中下载这些视频文件
3. 或者使用Git LFS存储大文件
