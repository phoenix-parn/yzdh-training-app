# 学习功能使用指南

## 📚 完整学习流程

### 第1步: 登录系统
1. 打开浏览器访问 `http://localhost:3001`
2. 输入测试账号:
   - 用户名: `admin`
   - 密码: `123456`
3. 点击"登录"按钮

### 第2步: 浏览课程
**在首页**:
- 可以看到3个工法课程卡片
- 每个卡片显示工法名称、进度、模块数量
- 点击任意课程卡片进入详情

**或者在课程页**:
- 点击底部导航栏的"课程"图标
- 查看所有课程列表
- 点击课程进入详情

### 第3步: 查看课程详情
**课程详情页包含**:
1. **顶部**: 课程名称和返回按钮
2. **视频区域**: 课程封面图(点击播放按钮可播放视频)
3. **学习进度**: 显示当前学习进度百分比
4. **模块列表**: 
   - 4个模块(基础认知、核心技术、实操流程、保障体系)
   - 点击模块标题可展开/收起
   - 每个模块下有多个小节

### 第4步: 进入知识点学习 ⭐
**这是核心学习功能**:

1. **展开模块**: 点击"基础认知模块"展开
2. **选择小节**: 点击任意小节,例如"1.1 工法背景与价值"
3. **进入知识点详情页**: 
   - 显示知识点ID徽章
   - 显示标题和所属模块
   - 显示详细文字内容
   - 显示相关图片素材
   - 显示可下载的文档资料

### 第5步: 查看学习统计
1. 点击底部导航栏的"我的"
2. 查看个人学习统计:
   - 学习次数
   - 总学习时长(实时累加)
3. 查看平台使用统计:
   - 总使用人数
   - 累计学习时长

## 🐛 常见问题排查

### 问题1: 看不到知识点内容
**原因**: 可能是JSON文件路径错误或内容为空

**解决方法**:
1. 打开浏览器开发者工具(F12)
2. 切换到Network标签
3. 点击小节进入知识点页面
4. 查看是否有404错误
5. 检查JSON文件是否存在:
   ```
   E:\CODE\SD-APP\public\course-materials\MF01\M01\content\1-1.1.1.json
   ```

### 问题2: 图片不显示
**原因**: 图片文件不存在或路径错误

**解决方法**:
1. 检查图片文件是否存在:
   ```
   E:\CODE\SD-APP\public\course-materials\MF01\M01\images\
   ```
2. 查看浏览器控制台是否有404错误
3. 确认图片路径格式正确

### 问题3: 点击小节没有反应
**原因**: 路由解析错误或知识点ID格式不对

**解决方法**:
1. 打开浏览器控制台查看错误信息
2. 确认知识点ID列表不为空
3. 检查App.tsx中的路由解析逻辑

### 问题4: 页面空白
**原因**: JavaScript错误或组件渲染失败

**解决方法**:
1. 打开浏览器控制台查看错误
2. 刷新页面重新加载
3. 清除浏览器缓存
4. 检查是否有TypeScript编译错误
