# 快速启动指南

## 第一次使用

### 1. 安装Node.js依赖
打开PowerShell,执行:
```powershell
cd E:\CODE\SD-APP
npm install
```

等待安装完成(可能需要几分钟)。

### 2. 启动开发服务器
```powershell
npm run dev
```

看到类似输出:
```
  VITE v6.3.5  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

### 3. 打开浏览器
访问: `http://localhost:5173`

### 4. 登录系统
使用任意测试账号登录:
- 用户名: `admin`
- 密码: `123456`

## 常见问题

### Q: npm install 失败?
A: 检查网络连接,或使用国内镜像:
```powershell
npm config set registry https://registry.npmmirror.com
npm install
```

### Q: 端口5173被占用?
A: 修改 `vite.config.ts`,更改端口号:
```typescript
server: {
  port: 3000  // 改为其他端口
}
```

### Q: 页面空白?
A: 打开浏览器控制台(F12),查看错误信息

### Q: 统计数据不更新?
A: 刷新页面,或清除浏览器缓存重新登录

## 开发说明

### 项目结构
```
E:\CODE\SD-APP/
├── public/
│   └── course-materials/    # 课程素材
├── src/
│   ├── components/          # React组件
│   ├── data/               # 课程数据
│   ├── utils/              # 工具函数
│   ├── App.tsx             # 主应用
│   └── main.tsx            # 入口文件
├── package.json            # 依赖配置
└── vite.config.ts          # Vite配置
```

### 热更新
修改代码后,浏览器会自动刷新,无需重启服务器。

### 停止服务器
在PowerShell中按 `Ctrl + C`
